# Elements of Information Theory

**Authors**: Thomas M. Cover, Joy A. Thomas
**Year**: 2006 (2nd edition)
**Publisher**: Wiley

**Used in Phase 1 for**: Entropy, cross-entropy, KL divergence, mutual information (Ch. 2)

**Key sections for ML foundations**:
- Ch. 2: Entropy, relative entropy, mutual information — the definitions that underlie every classification loss function
- Ch. 8: Differential entropy — for continuous distributions (Gaussian entropy, connection to MSE)

**Why it matters**: Every loss function in modern ML (cross-entropy, KL divergence, MSE under Gaussian assumption) is an information-theoretic quantity. Understanding the source derivations prevents cargo-cult loss function selection.
