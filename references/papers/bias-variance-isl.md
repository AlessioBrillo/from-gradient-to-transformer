# An Introduction to Statistical Learning (with Applications in Python)

**Authors**: Gareth James, Daniela Witten, Trevor Hastie, Robert Tibshirani
**Year**: 2023 (2nd edition, Python edition)
**Publisher**: Springer
**URL**: https://www.statlearning.com/

**Used in Phase 1 for**: Bias-variance tradeoff (Ch. 2.2), model assessment (Ch. 2.2.2)

**Key sections for foundations**:
- Ch. 2.2.2: Bias-Variance Decomposition — the derivation, the U-shaped test error curve, the tradeoff in practice
- Ch. 6: Regularization — ridge (L2) and lasso (L1) as bias-variance tools
- Ch. 7.1: Polynomial regression — the canonical example of bias-variance with varying complexity

**Why it matters**: ISL is the most accessible introduction to the bias-variance tradeoff with code examples. The polynomial regression simulation (Fig 2.9–2.11) is the best visual explanation of why more complexity eventually hurts.
